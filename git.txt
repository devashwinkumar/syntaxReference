-Clone a repository:
Git Clone

-Get latest changes from repository:
Git Pull

-Get the status of the local respostory:
git status

-Add changes to staging area:
git Add {filename}

--Commit the changes made with a message:
git commit -m"{meaningful message}"

--Add staged changes to the remote repositary:
git push 

--Initalise git repository:
git init

--Add everything in the current directory to staging area:
git add .

-- Add multiple files:
git add {file name} {file name}

-- Add alias name
git add {alias name} {respository url}
